Better Google Tasks

Have a Gmail account? Thought so.
Ever heard of Google Tasks?
You can now use it in a much better way.

Why use Google Tasks?

  • Free
  • Simple
  • Works with your Google account
  • Reliable cloud sync
  • Syncs with a huge selection of mobile apps

Why not?

  • No decent way to use it on a desktop

Until now


I used to use this

The standard Google tasks view is OK, but spending lots of time looking at it can make you feel sad inside.

I turned it into this

I took the exsisting view, removed a few things, improved a few things and generally made it more satisfying to get stuff done. It's not perfect, but it's better.

You can have it too. Here's how:

  1. Install the free Chrome Extension

    or go to https://mail.google.com/tasks/canvas
  3. Use and enjoy

Using Better Google Tasks

Help

Getting started

Once you've installed the Chrome extension click on the tasks icon, or go to https://mail.google.com/tasks/canvas. From now on every time you go to this address the extension will load the Better Google Tasks styles . I bookmark this link and set this as a pinned tab on Chrome so I always have it to hand. You can then type cmd + 1 to access it quickly.

Create a new task

Start typing to create a task and press enter when you're done.

Complete a task

Click the checkbox and magic will happen. I've tried to make it as satisfying as possible to check off tasks. It feels good to get stuff done.

Edit the task's details

Click arrow to the right of the task to edit it's details (or presss shift + enter). You can add some notes in the green area, add a due date or move to another list.

Indent a task

You can indent tasks by pressing tab, and un-indent by pressing shift and tab. This only works in the 'My order' view.

Create a new list

Click 'new list' at the bottom left of the page. A window will pop up where you can add a list name.

Delete or rename a list

Hover over a list item and you will see a pencil icon. Click on that and you can change the list name or delete it.

Show tasks in date order

When you've assigned dates to tasks it's helpful to view them in date order. Click 'sort by date' in the menu bar and that will happen. Pressing cmd/ctrl + up/down in this view will change the date, helpful if you're behind on due tasks.

Mobile apps

There are loads of Mobile apps that sync with Google Tasks and quite a few of them are good. I use Go Tasks for iphone, which is free. It's not super polished, but it works really well.

Keyboard shortcuts

  • Navigate tasks
    up arrow + down arrow
  • Move a task up the list
    cmd/ctrl + up
  • Move a task down the list
    cmd/ctrl + down
  • Indent task
    tab
  • Un-indent task
    shift + tab
  • Show task details
    shift + enter
  • Delete text
    delete
